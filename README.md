# Onlayn Mağaza

Müasir tələblərə uyğun onlayn mağaza layihəsi.

## Quraşdırma

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## İstifadə
- Frontend: `http://localhost:3000`
- Backend: `http://localhost:3001`
